### 支持功能：
* 防白嫖
    * 通过tg机器人一键验证
    * 接口处直接验证
* 简单接口直达车
    * 接口对接简单
    * 一条链接直达
* 后台可管理
    * 支持多群组授权对接
    * 支持对码库的管理
    * 多管理员管理
* 定时清除码库
